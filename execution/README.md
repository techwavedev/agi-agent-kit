# Execution Scripts

Deterministic Python scripts. See `_template.py`.

## Exit Codes

| Code | Meaning           |
| ---- | ----------------- |
| 0    | Success           |
| 1    | Invalid arguments |
| 2    | File not found    |
| 3    | API/Network error |
| 4    | Processing error  |

## Naming

Use `verb_noun.py` (e.g., `scrape_website.py`)
