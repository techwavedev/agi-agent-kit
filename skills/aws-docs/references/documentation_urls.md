## Usage with MCP Tools

### Reading a Documentation Page

```
read_documentation("https://docs.aws.amazon.com/IAM/latest/UserGuide/best-practices.html")
```

### Getting Recommendations

```
recommend("https://docs.aws.amazon.com/AmazonS3/latest/userguide/bucketnamingrules.html")
```

This returns related documentation pages that may be helpful for the topic at hand.
