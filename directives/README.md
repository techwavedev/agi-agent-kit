# Directives

SOPs written in Markdown that define what to do.

## Template

```markdown
# Directive: [Name]

> Last updated: YYYY-MM-DD

## Goal

[What this directive accomplishes]

## Inputs

- [Required input 1]

## Execution

1. Run `execution/script_name.py --args`

## Outputs

- [Deliverable and where it lives]

## Edge Cases

- **[Case]**: [How to handle]

## Learnings

- [YYYY-MM-DD] [What was learned]
```

## Naming

Use `lowercase_with_underscores.md`
